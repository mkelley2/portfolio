Epicodus Capstone: What's On?

  • Description: Do you watch too many tv shows and can't keep track of what's on? This app allows users to add their favorite tv shows to their account and automatically see what is on this week. Users can then scroll forwards and backwards through the weeks to see what they missed or what is coming up.
  • Technologies used: Drupal 7, JavaScript, PHP, pantheon, jQuery, TV maze API
  • Live site: http://dev-wutson.pantheonsite.io
  • Github Repo: https://github.com/mkelley2/wutson-pantheon

Homebase

  • Live site: http://dev-homebase.pantheonsite.io
  • Github Repo: https://github.com/mkelley2/Drupal-Homebase

DND Campaign Generator

  • Description: This website for the tabletop role-playing game Dungeons and Dragons allows a dungeon master the ability to generate a map, customize and create monsters and items, and set up character details to run a successful game. The dungeon master can save all pieces of the game they are currently playing and update it as their game goes on.

  • Github Repo: https://github.com/mkelley2/DnD-Campaign-Generator-and-Manager

Extreme Rock Paper Scissors

  • Description: This is personal project that was made completely for fun. It takes the age old Rock Paper Scissors and amplifies it. There are now 15 options to choose from. In the EXXTREME mode, when you lose a round the last move you made is removed from your options. The game goes until one person is completely out of moves. The game is available in both single and 2 player modes.
  • Technologies used: JavaScript, jQuery, PHP, Silex, Twig, CSS
  • Live site: http://extreme-rps.herokuapp.com/
  • Github Repo: https://github.com/mkelley2/ExtremeRPS

FEDCUS Trading System

  • Github Repo: https://github.com/mkelley2/FedCUS
